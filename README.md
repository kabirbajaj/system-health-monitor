# Automated System Health Monitoring Tool

This project is a Bash and Python-based system health monitoring tool designed to help Help Desk Technicians proactively identify common workstation issues.

## Features
- CPU, memory, disk usage monitoring
- System uptime tracking
- Network connectivity checks
- Automated report generation

## Technologies Used
- Bash scripting
- Python 3

## How to Run
1. chmod +x health_check.sh
2. ./health_check.sh
3. python3 health_report.py# system-health-monitor
I built a Bash and Python tool to automate system health checks, detect common workstation issues, and generate reports. The goal was to reduce repetitive help desk tickets and improve proactive troubleshooting.
